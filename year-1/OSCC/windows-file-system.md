# Windows File Systems

## FAT (File Allocation Table)
### Basics
- Used in flash media (USB drives)
- Versions:
	- FAT12
	- FAT16
	- FAT32
	- vFAT

### FAT Directory Entry
- Each file or directory are references and described in a separate directory entry


### Data Clusters
- Consist of one or more sectors
- Linked list
